# Real Time Chat App

A simple real time communication web application built using Python, Flask, and Socket.IO.

## Features
- Real time messaging
- Live communication
- Multiple connected users
- WebSocket communication
- Responsive interface

## Technologies
- Python
- Flask
- Flask-SocketIO
- HTML
- CSS
- JavaScript
- Git
- GitHub

## How It Works
Users connect to a Flask server through their browser. Messages are sent using WebSockets, allowing instant communication between connected users.
